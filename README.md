# 💸 Finance Buddy

**Finance Buddy** is a modern, secure, and user-friendly web platform that allows anyone to access and invest in private equity funds — regardless of their budget. Our mission is to democratize investment by making private equity more accessible, transparent, and manageable for all.

---

## 🚀 Features

- **User-Friendly Dashboard**  
  Track your investments and portfolio performance in real-time.

- **Secure Registration & Login**  
  Robust authentication system to protect your personal and financial data.

- **Investment Management**  
  View available private equity funds, manage contributions, and analyze your investments.

- **Admin Interface**  
  Admins can manage users, investment products, and monitor platform activity.

- **Scalable Architecture**  
  Designed with scalability and security in mind using modern web technologies.

## ⚙️ Getting Started

### Prerequisites

- Node.js
- Python 3.10+
- MySQL or PostgreSQL

🧰 Installation:
```
# Clone the repository
git clone https://github.com/solal10/Finance-buddy.git
cd Finance-buddy

# Install frontend dependencies
cd client
npm install

# Install backend dependencies
cd ../server
pip install -r requirements.txt
```

▶️ Running the App
```
# Start backend
cd server
python app.py

# Start frontend
cd ../client
npm start
```
